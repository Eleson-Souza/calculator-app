## Calculator App

Aplicação desenvolvida em React, trata de uma calculadora para cálculos aritméticos básicos (adição, subtração, multiplicação, divisão e porcentagem).

### Tecnologias :rocket:
* ReactJS
* JavaScript
* HTML
* CSS
